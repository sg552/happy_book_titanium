# 集合了所有的UI 演示的例子

下面这个操作务必做：

1. 下载官方的kitchensink到本地：

```
$ git clone https://github.com/appcelerator/KitchenSink.git
```

2. 编译。并运行。

Kitchensink, 是美国的一句俚语，表示基本都包含在里面了。

Appcelerator 的CEO 给项目起这个名字，估计跟他的军人经理有关。总之，这里
表示我们所有的UI 都可以在这里得到演示了。 无所不包的 DEMO。
